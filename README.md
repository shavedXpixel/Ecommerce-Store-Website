## 🌟 About The Project

Priyansu Store is a meticulously crafted e-commerce frontend designed to provide a premium shopping experience. Featuring curated product displays, smooth micro-animations, and a fully functional frontend cart and checkout system, this project showcases modern web development best practices.

### 🎨 Key Features

*   **✨ Modern & Premium Design:** Sleek interfaces built with Tailwind CSS and inspired by high-end brands.
*   **🛒 Interactive Shopping Cart:** Robust state management for adding items, adjusting quantities, and calculating totals in real-time.
*   **💳 Seamless Checkout Flow:** A beautifully designed checkout page that computes subtotals, shipping, and taxes dynamically.
*   **📱 Fully Responsive:** Flawless experience across all devices—from mobile screens to large desktop monitors.
*   **⚡ Blazing Fast Performance:** Powered by Vite for near-instant hot module replacement and optimized production builds.

## 🚀 Getting Started Locally

Follow these instructions to get a local copy up and running quickly.

### Prerequisites

You need Node.js installed on your machine. You can also use `bun` or `yarn` as your package manager.

### Installation

1. **Clone the repository**
   ```sh
   git clone <YOUR_GITHUB_REPO_URL>
   cd <YOUR_PROJECT_FOLDER>
   ```

2. **Install dependencies**
   Using npm:
   ```sh
   npm install
   ```
   Or using bun:
   ```sh
   bun install
   ```

3. **Start the development server**
   ```sh
   npm run dev
   ```
   Or with bun:
   ```sh
   bun run dev
   ```

4. **Open your browser**
   Navigate to the URL shown in your terminal (usually `http://localhost:5173`).

## 📂 Project Structure

A quick overview of the file structure:

```text
├── public/             # Static assets like images and icons
├── src/
│   ├── components/     # Reusable UI components (buttons, inputs, cards)
│   ├── pages/          # Individual page layouts (Home, Checkout, etc.)
│   ├── state/          # Global state management (Cart context/store)
│   ├── App.tsx         # Main application entry point
│   └── main.tsx        # React DOM rendering
├── tailwind.config.ts  # Tailwind CSS configuration
└── vite.config.ts      # Vite configuration
```

## ☁️ Deployment

Deploying this project is a breeze, especially on platforms like Vercel or Netlify.

### Vercel Deployment

1. Push your code to a GitHub repository.
2. Sign in to [Vercel](https://vercel.com/) and click **Add New... > Project**.
3. Import your GitHub repository.
4. Vercel automatically detects the Vite framework. The default settings are perfect:
   *   **Framework Preset:** Vite
   *   **Build Command:** `npm run build`
   *   **Output Directory:** `dist`
5. Click **Deploy**.

Within minutes, your site will be live on a `.vercel.app` domain!
